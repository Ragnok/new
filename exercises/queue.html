Queues

Assume that typedef queue stores double and has the following functions defined:

queue *queue_create(void)
void queue_destroy(queue *q)
void queue_enqueue(queue *q, double datum)
double queue_dequeue(queue *q)
bool queue_is_empty(queue *q)

Implement the following methods:

void queue_drain_into(queue *dst, queue *src)
Drains all items in src into dst.
void queue_flatten(queue **q)
Removes any duplicate items from the queue, retaining only the first occurrence of each: (1 1 5 2 2 3 3 2 3 3 5) becomes (1 5 2 3).

Challenge: Determine the Big-O complexity of each function.

Challenge: Implement versions where the data is of type void *.

Given the following implementation of typedef queue:

typedef struct {
	stack *enqueue;
	stack *dequeue;
} queue;

Implement the following functions:

queue *queue_create(void)
void queue_destroy(queue *q)
void queue_enqueue(queue *q, double datum)
double queue_dequeue(queue *q)
bool queue_is_empty(queue *q)
